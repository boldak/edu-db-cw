## Сценарії використання для користувача

- **_ID:_** USER.REG

- **_НАЗВА:_** Реєстрація в системі

- **_УЧАСНИКИ:_** Користувач, Система

- **_ПЕРЕДУМОВИ:_** Користувач не зареєстрований у системі

- **_РЕЗУЛЬТАТ:_** Новий обліковий запис

- **_ВИКЛЮЧНІ СИТУАЦІЇ:_** Відхилення запиту на реєстрацію (USER.REG_DENY)

- **_ОСНОВНИЙ СЦЕНАРІЙ:_**

@startuml

|Користувач|
start
:Вводить реєстраційні дані;

|Система|
:Перевіряє передані реєстраційні данія;
note right #ffaaaa
<b>Можлива</b>
<b>USER.REG_DENY</b>
end note

    :Cтворює новий обліковий запис, використовуючи введені дані;

    :Надає користувачу інформацію про створення облікового запису;

|Користувач|  
 :Завершує взаємодію з системою;

    stop;

@enduml
